**A2: Formulate the Equilibrium Equations**

- **Effective base normal force** (dry soil, \(u=0\)):
\[
N'_i = \frac{W_i + X_{i-1} - X_i + \dfrac{c'\,l_i}{F}\sin\alpha_i}
            {\cos\alpha_i - \dfrac{\tan\phi'}{F}\sin\alpha_i},
\qquad X_k = \lambda\,f(x_k)\,E_k.
\]

- **Force equilibrium recurrence** (horizontal direction, Spencer assumption \(f(x)=1\)):
\[
E_i = E_{i-1} - \frac{W_i K_i + \dfrac{c'\,l_i}{F}}{D_i + \lambda K_i},
\quad
K_i = \sin\alpha_i + \frac{\tan\phi'}{F}\cos\alpha_i,\;
D_i = \cos\alpha_i - \frac{\tan\phi'}{F}\sin\alpha_i.
\]
Boundary condition: \(E_0 = 0\); force equilibrium requires \(E_5 = 0\).

- **Moment equilibrium about slip centre** \((95,120)\):
\[
\sum_{i=1}^{5} W_i\,(x_{\text{mid},i} - 95) \;-\; R\sum_{i=1}^{5} S_i = 0,
\qquad
S_i = \frac{c'\,l_i + N'_i \tan\phi'}{F}.
\]

- **Transfer coefficient** (links slice \(j\) to \(j+1\)):
\[
\Psi_{i-1} = \frac{D_i - \lambda f_{i-1} K_i}{D_i - \lambda f_i K_i}.
\]
With \(f(x)=1\), \(\Psi_{i-1} \equiv 1\).

**PART B — METHODOLOGICAL VERIFICATION**

**B1: Angle Convention**
\[
\alpha_i = \arctan\!\left( \frac{x_{\text{mid},i} - 95}{120 - y_{\text{bot},i}} \right).
\]
- Active zone (\(x_{\text{mid}} < 95\)): \(\alpha_i < 0\) (negative).
- Passive zone (\(x_{\text{mid}} > 95\)): \(\alpha_i > 0\) (positive).

**B2: Solver Strategy**
The explicit recurrence relations given in A2 were used. \(F\) was adjusted by a secant method on the force residual \(E_5(F,\lambda)=0\); \(\lambda\) was adjusted by a secant method on the moment residual \(\sum W_i(x_{\text{mid},i}-95) - R\sum S_i = 0\). No general‑purpose non‑linear root finder was employed.

**B3: Base Normal Force**
\[
N'_i = \frac{W_i + X_{i-1} - X_i + \dfrac{c'\,l_i}{F}\sin\alpha_i}
            {\cos\alpha_i - \dfrac{\tan\phi'}{F}\sin\alpha_i},
\qquad
X_k = \lambda\,E_k \quad (\text{since } f(x)=1).
\]
The formulation explicitly includes the interslice shear forces \((X)\) and, through the recurrence, the interslice normal forces \((E)\).

**B4: Transfer Coefficient (\(\Psi\))**
\[
\Psi_{i-1} = \frac{D_i - \lambda f_{i-1} K_i}
               {D_i - \lambda f_i K_i},
\quad
K_i = \sin\alpha_i + \frac{\tan\phi'}{F}\cos\alpha_i,\;
D_i = \cos\alpha_i - \frac{\tan\phi'}{F}\sin\alpha_i.
\]
With \(f(x)=1\) (Spencer), \(\Psi_{i-1}=1\).

**B5: Interslice Function Indexing**
For slice \(k\), the interslice function \(f(x)\) is evaluated at the left vertical boundary \(x = x_{k-1}\) and at the right vertical boundary \(x = x_k\). These values multiply the corresponding interslice normal forces:
\[
X_{k-1} = \lambda\, f(x_{k-1})\, E_{k-1}, \qquad
X_k     = \lambda\, f(x_k)\, E_k.
\]
