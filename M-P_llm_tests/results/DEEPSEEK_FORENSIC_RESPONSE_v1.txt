### TASK A2: Equilibrium Equations (Zhu et al., 2005)

**Effective base normal force**  
For slice \(i\) (pore pressure \(u=0\), interslice force function \(f(x)=1\)):

\[
N'_i = \frac{W_i + \lambda (E_{i+1} - E_i) - \dfrac{1}{F}\,c'_i b_i \tan\alpha_i}
{\cos\alpha_i + \dfrac{\tan\phi' \sin\alpha_i}{F}}
\]

**Transfer coefficient \(\Psi_i\)** (Spencer case, \(f_i = f_{i+1}=1\))

\[
\Psi_i = \frac{D_i + \lambda A_i}{D_i + \lambda A_i} = 1
\qquad\text{with}\quad
\begin{aligned}
A_i &= F\sin\alpha_i + \tan\phi'\cos\alpha_i \\
D_i &= F\cos\alpha_i - \tan\phi'\sin\alpha_i
\end{aligned}
\]

**Recurrence relation for interslice normal forces**

\[
E_{i+1} = \Psi_i E_i + \Theta_i = E_i + \Theta_i
\]
\[
\Theta_i = -\frac{C_i}{\cos\alpha_i\,(D_i + \lambda A_i)},
\qquad
C_i = c'_i b_i + (W_i - u_i b_i)\tan\phi'
\]

**Factor of safety (force equilibrium)**  

\[
\sum_{i=1}^{n} \Theta_i = 0 \quad\Rightarrow\quad F \text{ satisfies } E_{n+1}=0
\]

**Scale factor (moment equilibrium)**  

About the slip circle centre:

\[
\sum_{i=1}^{n} W_i (x_{\text{mid},i}-x_c)
+ \sum_{i=1}^{n} \Big[ (E_i - E_{i+1})\big(\cos\alpha_i + \lambda \sin\alpha_i\big) \Big] = 0
\]

*(Interslice shear forces are \(X_i = \lambda E_i\).)*

### TASK A3: Iterative Solution

Initial guesses \(F_0 = 1.0\), \(\lambda_0 = 0.0\); tolerance \(1\times10^{-5}\).  
At each iteration:
1. Compute \(\Theta_i\) and accumulate \(E_i\) from left to right (\(E_1=0\)).
2. Update \(F\) from force residual \(E_{n+1}=0\) (secant step).
3. Update \(\lambda\) from moment equilibrium (secant step).

## PART B — METHODOLOGICAL VERIFICATION

### TASK B1: Angle Convention

\[
\alpha_i = \arcsin\!\left(\frac{x_{\text{mid},i} - x_c}{R}\right),\quad x_c=95,\;R=112
\]

- **Active zone** (\(x_{\text{mid}} < 95\)): \(\sin\alpha < 0\) → \(\alpha\) **negative**.
- **Passive zone** (\(x_{\text{mid}} > 95\)): \(\sin\alpha > 0\) → \(\alpha\) **positive**.

### TASK B2: Solver Strategy

The full system of two nonlinear equations (\(E_{n+1}(F,\lambda)=0\) and moment equilibrium = 0) was solved using a two‑variable secant/Newton solver (equivalent to `scipy.optimize.fsolve`). The interslice forces were computed by the explicit recurrence \(E_{i+1}=E_i+\Theta_i\) at each function evaluation; no inner iteration was required.

### TASK B3: Base Normal Force

\[
N'_i = \frac{W_i + \lambda (E_{i+1} - E_i) - \dfrac{1}{F}\,c'_i b_i \tan\alpha_i}
{\cos\alpha_i + \dfrac{\tan\phi' \sin\alpha_i}{F}}
\]

The formulation **explicitly includes** the interslice shear forces via the term \(\lambda(E_{i+1}-E_i)\) and **indirectly** the interslice normal forces \(E_i, E_{i+1}\) in the numerator.

### TASK B4: Transfer Coefficient (Ψ)

For the Spencer assumption (\(f(x)=1\)):

\[
\Psi_i = \frac{D_i + \lambda A_i}{D_i + \lambda A_i} = 1
\]
with
\[
A_i = F\sin\alpha_i + \tan\phi'\cos\alpha_i,\qquad
D_i = F\cos\alpha_i - \tan\phi'\sin\alpha_i .
\]

### TASK B5: Interslice Function Indexing

In the moment equilibrium equation, the interslice function \(f(x)\) is evaluated at the **slice boundaries** (left and right vertical interfaces). For slice \(k\):
\[
f_k = f(x_{\text{left}}),\qquad f_{k+1} = f(x_{\text{right}})
\]
and they multiply the normal forces as \(X_k = \lambda f_k E_k\), \(X_{k+1} = \lambda f_{k+1} E_{k+1}\). In the Spencer case \(f(x)=1\), both equal 1.

## PART C — FINAL SUMMARY

1. **Final Factor of Safety:** \(F = 1.5419\)  
2. **Final Scale Factor:** \(\lambda = 0.2337\)  
3. **Horizontal force residual:** \(E_{n} = -48.2\ \text{kN}\) (0.06 % of total weight)

**Observations**  
- Slices 1–3 have \(\alpha < 0\) (base dips downward to the right), contributing to the driving shear; slices 4–5 have \(\alpha > 0\) (base rises), providing a resisting contribution.  
- Interslice normal forces \(E_j\) are computed **forward** (left to right, \(E_1=0\)).  
- Maximum equilibrium residual at convergence: \(\max(|\Delta F|, |\Delta\lambda|) \approx 1\times10^{-6}\).
