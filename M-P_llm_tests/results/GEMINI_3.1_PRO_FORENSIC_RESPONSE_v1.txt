#### TASK A2: Formulate the Equilibrium Equations

**1. Effective Normal Base Force ($N'_i$):**
$$N'_i = \frac{W_i - \Delta X_i - \frac{c' b_i \tan\alpha_i}{F}}{m_{\alpha i}}$$
Where $\Delta X_i = \lambda f_i E_i - \lambda f_{i-1} E_{i-1}$ and $m_{\alpha i} = \cos\alpha_i + \frac{\sin\alpha_i \tan\phi'}{F}$.

**2. Zhu et al. (2005) Explicit Equations:**
*   **Factor of Safety ($F$) from Force Equilibrium:**
    $$F = \frac{ \sum_{i=1}^n \left[ \frac{c' b_i + (W_i - \Delta X_i)\tan\phi'}{m_{\alpha i} \cos\alpha_i} \right] }{ \sum_{i=1}^n (W_i - \Delta X_i) \tan\alpha_i }$$
*   **Scale Factor ($\lambda$) from Moment Equilibrium:**
    $$\lambda = \frac{ \sum_{i=1}^n \left[ W_i \sin\alpha_i - \frac{c' b_i + W_i \tan\phi'}{F m_{\alpha i}} \right] }{ \sum_{i=1}^n \Delta x_i \left[ \sin\alpha_i - \frac{\tan\phi'}{F m_{\alpha i}} \right] }$$
    *(where $\Delta x_i = f_i E_i - f_{i-1} E_{i-1}$)*

**3. Transfer Coefficient ($\Psi_j$):**
$$E_{i} = \Psi_{i-1} E_{i-1} + \Phi_i$$
$$\Psi_{i-1} = \frac{ \cos\alpha_i \left(1 - \lambda f_{i-1} \frac{\tan\phi'}{F}\right) + \sin\alpha_i \left(\lambda f_{i-1} + \frac{\tan\phi'}{F}\right) }{ \cos\alpha_i \left(1 - \lambda f_i \frac{\tan\phi'}{F}\right) + \sin\alpha_i \left(\lambda f_i + \frac{\tan\phi'}{F}\right) }$$

### PART B — METHODOLOGICAL VERIFICATION

#### TASK B1: Angle Convention
**Formula:** $\alpha = \arcsin\left(\frac{x_c - x_{mid}}{R}\right)$
**Sign Evaluation:** $\alpha$ evaluates to a **positive** value in the active (driving) zone ($x_{mid} < x_c$) and a **negative** value in the passive (resisting) zone ($x_{mid} > x_c$). 

#### TASK B2: Solver Strategy
I used the **exact explicit recurrence relations** derived by Zhu et al. (2005) executed via a fixed-point iterative scheme updating $F$ and $\lambda$ at each step, rather than utilizing a general-purpose Jacobian non-linear root finder.

#### TASK B3: Base Normal Force
**Formula:** 
$$N'_i = \frac{W_i - \Delta X_i - \frac{c' b_i \tan\alpha_i}{F}}{\cos\alpha_i + \frac{\sin\alpha_i \tan\phi'}{F}}$$
**Explicit Inclusion:** The formula explicitly includes the interslice shear forces ($X$) as $\Delta X_i = X_R - X_L$. It implicitly includes the interslice normal forces ($E$) by definition because $\Delta X_i = \lambda f_i E_i - \lambda f_{i-1} E_{i-1}$.

#### TASK B4: Transfer Coefficient (Ψ)
To link slice $j$ and slice $j+1$ (transitioning from $E_j$ to $E_{j+1}$):

**Numerator:**
$\cos\alpha_{j+1} \left(1 - \lambda f_j \frac{\tan\phi'}{F}\right) + \sin\alpha_{j+1} \left(\lambda f_j + \frac{\tan\phi'}{F}\right)$

**Denominator:**
$\cos\alpha_{j+1} \left(1 - \lambda f_{j+1} \frac{\tan\phi'}{F}\right) + \sin\alpha_{j+1} \left(\lambda f_{j+1} + \frac{\tan\phi'}{F}\right)$

#### TASK B5: Interslice Function Indexing
For a given slice $k$, the function is evaluated at the **left vertical boundary** as $f_{k-1}$ (which multiplies the incoming normal force $E_{k-1}$) and at the **right vertical boundary** as $f_k$ (which multiplies the outgoing normal force $E_k$).
